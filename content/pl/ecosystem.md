# Pakiety ekosystemu

| Pakiet | Rola |
|---|---|
| **koru** | Pętla: kolejka, cykle, orkiestracja przejazdów, dashboard, doctor. |
| **planfile** | Tickety, sprinty, kolejka — kręgosłup audytu. |
| **tillm** | Rejestr i sterowniki klientów shell-LLM (zero zależności; twarda zależność koru). |
| **gillm** | Wstrzykiwanie GUI: sterowniki klawiatury/myszy, klasyfikacja awarii. |
| **koruide** | Tożsamość i control-plane IDE: detekcja, aliasy, protokół daemona pluginów. |
| **vdisplay** | Prawda o ekranie: zrzuty, mapy elementów VQL, zweryfikowana aktuacja GUI. |
| **regix / testql / wup** | Bramki jakości, zachowania i on-change. |
| **goal** | Automatyzacja repo: deps → diagnoza → testy → commit/publikacja. |
