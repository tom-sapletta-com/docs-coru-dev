# Ecosystem packages

| Package | Role |
|---|---|
| **koru** | The loop: queue, cycles, drive orchestration, dashboard, doctor. |
| **planfile** | Tickets, sprints, queue — the audit backbone. |
| **tillm** | Shell-LLM client registry & drivers (zero-dependency; a core dependency of koru). |
| **gillm** | GUI injection: keyboard/mouse drivers, recovery classification. |
| **koruide** | IDE identity & control plane: detection, aliases, plugin daemon protocol. |
| **vdisplay** | Screen truth: captures, VQL element maps, verified GUI actuation. |
| **regix / testql / wup** | Quality, behavior, and on-change gates. |
| **goal** | Repo automation: deps → diagnose → test → commit/publish. |
